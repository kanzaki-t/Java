- 子类可以获取到父类中声明的所有的属性和方法
- 但是，由于封装性的影响，可能子类不能直接调用父类中声明的属性或方法，但是可以用setxxx方法更改属性
- 子类在继承父类以后，还可以扩展自己特有的功能（增加特有的属性或方法）
- 获取父类的方法
    .getClass().getSuperclass()
- 默认的父类
    java中声明的类，如果没有显式的声明其父类时，则默认继承java.lang.Object
- java时支持多层继承
    直接父类，间接父类
- java中的子父类的概念时相对的
- java中一个父类可以声明多个子类，反之，一个子类只能有一个父类，Java的单继承性
