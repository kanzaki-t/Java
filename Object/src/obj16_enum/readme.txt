- 枚举类
    枚举类本是上也是一种类，只不过这个类的对象是有限的，固定的几个，不能让用户随意创建
    例子：
        星期，月，季节...

    开发中的建议：
        如果针对于某个类，其实例是确定个数的，则推荐将此类声明为枚举类

    使用Enum声明
    默认其父类是 java.lang.Enum

    实现接口：
        1.枚举类实现接口，在枚举类中重写接口中的抽象方法，
        当通过不同的枚举类对象调用此方法时，执行的是同一个方法

        2.让枚举类的每一个对象重写接口中的抽象方法，
        当通过不同的枚举类对象调用此方法时，执行的是不同的实现类的方法
