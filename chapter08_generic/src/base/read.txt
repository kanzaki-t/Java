    自定义泛型类

        }

    自定义泛型接口
        interface B<T>{
        }

    ORM思想（object relational mapping）
        > 数据库中的一个表 与 Java中的一个类对应
        > 表中的一条记录 与 Java类的一个对象对应
        > 表中的一个字段或者列 与 Java类的对象的一个属性对应
